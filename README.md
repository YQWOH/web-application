## Setup Instructions
To run on local computer:
`git clone https://github.com/YQWOH/web-application.git`

### Production Site:
[https://celadon-ganache-136900.netlify.app/]

### Frameworks Used: 
- [x] HTML
- [x] CSS
- [x] Javascript
- [x] Vue Js
- [x] Tailwind CSS
- [x] Pinia

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
